# JUnit Example

## Scope

The project consist of a single utility class [Palindromes](./src/main/java/hu/aestallon/example/Palindromes.java) to determine whether a string is a palindrome or not.

## How to Run Tests

Standing in the project root, execute

- On Windows:
  ```shell
  .\gradlew test
  ```
- On Linux/macOS:
  ```shell
  ./gradew test
  ```
